<Warning>
  Rainy MaTE is currently in `BETA 1`.
</Warning>

## Current availability

- Supported OS in this release: `macOS`.
- `Windows` and `Linux` support is planned for upcoming releases.

## Important beta behavior

- This build does not yet include Apple code signing/notarization.
- On first launch, macOS security may block the app until you allow it manually.
- See `/users/install-desktop` for the exact allow steps.

## Feature stability policy

Features in BETA are not static.

- Behavior can change between releases.
- UI labels and workflows can be adjusted.
- Some capabilities may be replaced, renamed, or removed.

## What this means for teams

- Review release notes before each upgrade.
- Validate critical workflows in a staging workspace first.
- Avoid hard dependencies on undocumented behavior.
