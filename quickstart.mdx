## Prerequisites

- A supported desktop OS: macOS, Windows, or Linux.
- One provider API key for either `Rainy API` or `Google Gemini`.
- A local folder you want Rainy MaTE to work in.

## Install the desktop app

1. Open the [Rainy MaTE releases page](https://github.com/ferxalbs/rainy-cowork/releases).
2. Download the installer for your operating system.
3. Install and open the app.

## Configure your provider key

1. Open `AI Provider` in the sidebar.
2. Paste your API key for `Rainy API` or `Google Gemini`.
3. Run `Validate`, then `Save`.

<Info>
  Keys are stored locally by the desktop app. You can view, replace, or delete
  them from the same settings page.
</Info>

## Select a workspace folder

1. In the `Workspace` section, click `Add First` or `+`.
2. Choose the project folder where the agent should work.
3. The selected folder becomes the active workspace context.

## Run your first task

1. Open `Agent Chat`.
2. Select a model and an agent from the top controls.
3. Send a prompt, for example: `Summarize this repository structure and list the highest-risk files to review first.`
