## Prerequisites

Before installing Rainy MaTE, ensure you have the following:

- **Operating System**: macOS 12+ (Monterey or later), Windows 10/11, or Linux (Ubuntu 20.04+).
- **Hardware**: Minimum 8GB RAM (16GB recommended for local LLMs).
- **API Keys**: An OpenAI or Anthropic API key is required for cloud-based intelligence.

## Installation

1.  **Download the Installer**: Visit our [releases page](https://github.com/ferxalbs/rainy-cowork/releases) and download the appropriate installer for your OS.
2.  **Run the Installer**: Follow the on-screen instructions to install Rainy MaTE.
3.  **Launch the App**: Open Rainy MaTE from your applications folder.

## Initial Configuration

On first launch, you will be guided through a setup wizard:

1.  **Sign In**: Create an account or sign in to sync your preferences.
2.  **Connect AI Provider**: Enter your API key (e.g., OpenAI, Anthropic).
3.  **Grant Permissions**: Rainy MaTE needs access to your file system to perform tasks. Grant the necessary permissions when prompted.

## Your First Task

Once installed, try asking Rainy MaTE to perform a simple task:

1.  Open the **Neural Panel** (`Cmd + .`).
2.  Type: _"Analyze the current directory and summarize the project structure."_
3.  Watch as Rainy MaTE autonomously explores your files and generates a report.
