## Purpose

`Rainy ATM` is the cloud operations panel for connecting a desktop node to Cloud Cortex and managing remote execution.

## Connect the node

1. Open `Rainy ATM` in the sidebar.
2. Enter `Platform key` and `User API key`.
3. Optionally set `Workspace name`.
4. Confirm connection status in Dashboard.

## Dashboard tab

- Shows workspace identifiers.
- Links directly to Cloud Cortex.
- Generates temporary session codes for Telegram pairing.

## Agents tab

- Shows active deployed agents.
- Supports fast navigation to Agent Builder for updates.

## Activity tab

- Lists recent command runs.
- Shows status and timing data.
- Helps investigate delays and failed executions.

<Info>
  Use Activity as your primary audit trail for production remote workflows.
</Info>
